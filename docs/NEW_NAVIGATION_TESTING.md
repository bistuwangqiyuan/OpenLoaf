# 新导航系统测试指南

## 启用新导航系统

### 1. 设置环境变量

在项目根目录创建或修改 `.env.local` 文件：

```bash
NEXT_PUBLIC_USE_NEW_NAVIGATION=true
```

### 2. 重启开发服务器

```bash
pnpm run dev:web
```

## 功能测试清单

### ✅ 基础导航测试

- [ ] **工作台视图**
  - 点击侧边栏"工作台"按钮
  - 验证页面标题显示"工作台"
  - 验证左侧显示工作台内容

- [ ] **日历视图**
  - 点击侧边栏"日历"按钮
  - 验证页面标题显示"日历"
  - 验证左侧显示日历内容

- [ ] **邮件视图**
  - 点击侧边栏"邮箱"按钮
  - 验证页面标题显示"邮箱"
  - 验证左侧显示邮件列表
  - 验证未读邮件数量显示

- [ ] **任务视图**

### ✅ AI 助手和对话测试

- [ ] **创建新对话**
  - 点击侧边栏"AI 助手"按钮
  - 验证创建新的对话会话
  - 验证页面显示 Chat 界面
  - 发送一条消息，验证对话正常工作

- [ ] **Workspace 对话列表**
  - 创建多个对话（点击多次"AI 助手"）
  - 验证对话列表显示在侧边栏项目树下方
  - 验证对话按更新时间排序（最新的在前）
  - 验证默认显示 10 条对话

- [ ] **对话列表交互**
  - 点击对话列表中的对话，验证切换成功
  - 验证页面标题显示对话标题
  - 右键点击对话，验证菜单显示
  - 测试"转为项目"功能
  - 测试"删除"功能

### ✅ 对话转项目测试

- [ ] **转换流程**
  - 创建一个对话并发送几条消息
  - 右键点击对话，选择"转为项目"
  - 在对话框中输入项目名称
  - 可选：选择父项目
  - 点击"转换"按钮

- [ ] **转换验证**
  - 验证项目出现在侧边栏项目树中
  - 验证对话从 Workspace 对话列表中消失
  - 验证项目目录包含对话文件
  - 验证数据库 ChatSession.projectId 已更新

### ✅ 布局和交互测试

- [ ] **面板宽度调整**
  - 在有左右面板的视图中（如项目视图）
  - 拖拽中间的分隔线
  - 验证左侧面板宽度实时调整
  - 验证宽度限制（最小宽度 300px）
  - 验证拖拽时鼠标样式变化

- [ ] **视图切换**
  - 在不同视图间快速切换

- [ ] **响应式布局**
  - 调整浏览器窗口大小
  - 验证布局自适应
  - 验证窄屏时侧边栏隐藏

## 已知限制

### 当前版本不支持的功能

1. **项目视图**
   - 项目点击逻辑尚未实现
   - 需要在 SidebarProject.tsx 中添加支持

2. **URL 路由同步**
   - 视图切换不会更新 URL
   - 刷新页面会丢失当前视图状态

3. **浏览器历史**
   - 前进/后退按钮不会切换视图

4. **Stack 面板**
   - Stack 面板管理尚未完全实现
   - 面板堆叠功能需要进一步开发

5. **对话历史加载**
   - 对话列表的历史消息加载需要优化
   - 分页加载尚未实现

## 错误处理测试

- [ ] **网络错误**
  - 断开网络连接
  - 尝试创建对话
  - 验证错误提示

- [ ] **无效操作**
  - 尝试删除不存在的对话
  - 尝试转换已关联项目的对话
  - 验证错误提示

## 回退到旧系统

如果遇到问题，可以随时回退到旧的 Tab 系统：

1. 删除或注释 `.env.local` 中的 `NEXT_PUBLIC_USE_NEW_NAVIGATION=true`
2. 重启开发服务器
