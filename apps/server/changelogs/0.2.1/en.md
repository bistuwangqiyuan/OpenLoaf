## New Features

- AI Agent prompt multilingual support (10 templates with bilingual prompts)
- agentFactory multilingual infrastructure (runtime prompt language selection)
- tRPC error message internationalization framework

- Workspace default names multilingual support (8 languages predefined)
- Tab title support for titleKey internationalization

- Add zh-TW to uiLanguage whitelist in settings service
- Migrate calendar event form UI strings to i18next

- Fix encoding issues and multilingual prompt exports
- Enhance error handling and internationalization in API procedures
